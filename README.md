# 状态
## 正常
 - 键盘、触摸板 (GPIO中断、支持手势)
 - 雷雳、 USB部分正常、PCIe隧道无设备测试、视频输出直通独显无解
 - HDMI、直通独显无解
 - Wi-Fi 蓝牙 (驱动自行添加 EFI中不包含无线网卡驱动)
 - 声卡 (Intel SST 内置麦克风无法驱动、耳机需要ALCFixPlug,否则无法使用->[点我下载](https://github.com/Core-i99/DELL-G15-5510-ALCPlugFix))
 - 亮度 
 - 快捷键(部分)
 - 电池
 - 睡眠 (S3 支持)
 - USB
## 不正常
 - 帮别人做, 没详细测试, 有问题请提交issues
